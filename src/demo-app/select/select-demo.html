 Keyboard Navigation deja-select has some keyboard navigation feature
  • UpArrow: Select the previous item
  • DownArrow: Select the next item
  • Alt+UpArrow: Toogle the dropdown
  • Alt+DownArrow: Toogle the dropdown
  • PageUp: Select the next page item
  • PageDown: Select the previous page item
  • Home: Select the first item
  • End: Select the last item
  • Space: Collapse or expand a group header
  • Enter: Select the current item and close the dropdown
  • A-Z 0-9: Select the next item started with the pressed key
